# Mirkwood Sprite Forge

一个浏览器端精灵图动画工坊，用于把 sprite sheet 按网格或自由框切成动画帧，并预览、抠除背景、导出 GIF、PNG、CSS keyframes 和 JSON 元数据。

## 使用

启动本地前端服务：

```bash
npm run dev
```

然后打开终端输出的地址，默认是：

```text
http://127.0.0.1:5173
```

不需要安装依赖，服务由 Node 内置模块提供。

1. 拖入或选择一张精灵图。
2. 严格网格图使用“网格”模式，调整帧宽、帧高、边距、间距、行列数。
3. AI 生成的非均匀图使用“自由框”模式，点“智能识别”生成帧框。
4. 识别不准时，在画布上拖拽添加帧，点击帧框选中后可删除。
5. 如果两个动作被识别进同一个大框，选中大框后点“纵向拆分”或“横向拆分”；按住 Alt/Option 点击大框可按点击位置纵向拆分，Shift + Alt/Option 点击可横向拆分。
6. 需要透明背景时，打开“导出透明”。背景不干净时调“抠图阈值”，或点“取背景点”后在画布背景上取样。
7. 设置起始帧、帧数量、FPS、播放方向。
8. 导出 GIF、横向序列、逐帧 PNG、当前帧、CSS 或 JSON。

导入图片后默认暂停，确认切片范围后再点击“播放”预览动画。

## 文件

- `index.html`：工具界面
- `styles.css`：布局和视觉样式
- `app.js`：切帧、预览、导出逻辑
- `server.mjs`：本地前端服务
